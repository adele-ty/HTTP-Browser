# 发送url到页面中，发生了什么
1. DNS 解析:将域名解析成 IP 地址  
2. TCP 连接：TCP 三次握手  
3. 发送 HTTP 请求  
4. 服务器处理请求并返回 HTTP 报文  
5. 浏览器解析渲染页面  
6. 断开连接：TCP 四次挥手
